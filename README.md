
# 📦 SpringGrocery - Organic Store (Java web page with Servlets & JSP, Hibernate)

**springweb1** is a Java web application built using **Servlets**, **JSP**, and **Hibernate**.
It simulates an online grocery store with product categories, shopping cart, user login/registration, and order history.

---

## ✅ Features

- 📋 **User Registration & Login**
- 🛒 **Shopping Cart Management**
- 🛍️ **Product Categories & Item Listing**
- 📜 **Order History**
- 🔐 **Session Management & Logout**
- 🔗 **Hibernate Integration-Updated in MySQL**
- 🎨 **Simple JSP-based Frontend**

---

## ⚙️ Technologies Used

- Java Servlets & JSP
- ORM Hibernate 6 
- Database: MySQL 
- Apache Tomcat 10 - Web Server(for deployment)

---

## 🚀 How to Run

1. Create a database named springgrocery
2. Run the Tomcat Server
3. Access the app in your browser:
   ```
   http://localhost:8080/springweb1/
   ```
